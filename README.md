## 🍀 목표
- GraphQL을 활용한 영화 정보 API 구축
- 클라이언트와 서버 간의 효율적인 데이터 통신 구현


## ❗️ Key Point
- GraphQL 스키마 설계
- 영화 데이터를 가져오는 외부 API 연동
- 클라이언트 측에서 원하는 데이터만 선택적으로 가져올 수 있음

## 💻 실행방법
```
npm install

//server 실행
node server.js

//localhost 주소로 이동
http://localhost:4000/
```

## 📝 주요기능
- 영화 목록 조회
- 특정 영화의 상세 정보 조회
- 사용자의 리뷰 및 평점 조회
